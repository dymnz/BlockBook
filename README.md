
# Setup
## TestRPC Setup
```
$ install nodejs/npm
$ apt-get install build-essential python
$ npm install ethereumjs-testrpc -g
$ npm install web3 -g
$ testrpc -u 0
```

## Truffle Setup
```
$ npm install webpack -g
$ npm install truffle -g
$ truffle init webpack
$ truffle compile
$ truffle migrate
```

## After Pulling
```
$ npm install
$ truffle compile
$ truffle migrate
```
